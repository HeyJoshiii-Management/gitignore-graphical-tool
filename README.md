# HeyJoshiii Gitignore Graphical Tool

Entwicklung des grafischen Tools zum generieren der Gitignore-Dateien für unsere Git-Repositories des Minecraft Server-Deployments.

Die Idee hinter dem Tool ist es, die gitignore-Dateien durch Automatismen, zuverlässig und funktionierend, zu generieren.
Insbesondere mit Hinblick auf Negationen in gitignore-Dateien sowie der schieren masse an Pfaden und Dateien welche bei Minecraft-Server stets wild verteilt sind.
Dabei soll das Tool automatisch durch die Server "laufen" und sämtliche Dateipfade und Dateien listen. Basierend dieser Listung, kann ausgewählt werden, was in der gitignore auftauchen soll.
